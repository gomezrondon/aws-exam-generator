# aws-exam-generator
A Kotlin Spring boot program that generate AWS certification questions.

### To run the program
 ```
gradle clean build -x test
java -jar build/libs/aws-exam-generator-0.0.1-SNAPSHOT.jar
 ```
select option (1)
then select the number of questions, example (5)
